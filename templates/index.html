Cody

An online code runner API built with Flask + Docker.

🎯 Live Code Editor

📡 Sample Request (cURL)

curl -X POST http://localhost:5000/api/v1/cody \
  -H "Content-Type: application/json" \
  -d '{"language": "python", "code": "print(\"Hello, Cody!\")"}'

✅ Supported Languages

  • Python
  • JavaScript (Node.js)
